# Student Performance Analysis

## Overview
This project analyzes student performance data using SQL, Excel, and Python. It focuses on understanding how students perform across subjects and departments. It also studies the relationship between attendance and marks. The goal is to generate simple insights that can help improve academic performance and support students.

---

## Project Structure

- **sql/** → Contains SQL script and query outputs  
- **excel/** → Contains Excel file and pivot table output  
- **python/** → Contains dataset, notebook, and visualizations  

---

## Tech Stack

- SQL (MySQL Workbench)  
- Microsoft Excel (WPS Office)  
- Python (Google Colab)  
- Libraries: Pandas, Matplotlib  

---

## Features

- Calculate average marks for students  
- Identify top-performing students  
- Compare department performance (CS vs IT)  
- Detect students with low marks  
- Analyze impact of attendance on performance  
- Visualize data using charts  

---

## SQL Analysis

SQL is used to create and manage the database. It stores student data and runs queries to calculate average marks, find top students, and compare departments. It also helps identify students with low marks.

---

## Excel Analysis

Excel is used to calculate average marks, sort students, and highlight low scores. A pivot table is created to show department-wise performance. It makes the data easy to understand without coding.

---

## Python Analysis

Python is used for deeper analysis using Pandas. It helps in grouping data, calculating averages, and finding the top student. It also checks the relationship between attendance and marks.

---

## Visualization

- Bar chart → Student average marks  
- Column chart → Department performance  
- Scatter plot → Attendance vs marks  

These graphs help in understanding trends clearly.

---

## Key Insights

- Some students perform significantly better than others  
- CS and IT departments show different performance levels  
- Students with higher attendance tend to perform better  
- Few students are at risk due to low marks
